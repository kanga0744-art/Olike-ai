# Olike-ai
Web image generator, with pollination base. Equipped with the latest model updates.
Olike - AI Image Generator
An intuitive and fast generation AI image platform, powered by cutting-edge models from Pollinations. Produce high-quality images from text (T2I), convert images based on instructions (I2I), to analyze images into descriptive prompt (I2P).

Main Features:

Text-to-Image (T2I): Generate images from a text prompt more controlled results.
Image-to-Image (I2I): Change input image based on prompt and guide strength (strength), support multi-input.
Image-to-Prompt (I2P): Image analysis into text description, can be used as a prompt for T2I or I2I.
AI Assistant: Help AI to design a prompt or remix an existing prompt with one click.
Private Gallery: Save image results to local storage, complete with metadata and basic editing tools (brightness, contrast, rotation, crop).
Pollen system: Monitoring tokens with daily reset.

Supported Models:
Flux, FLUX.2 Klein (4B & 9B), Zimage, Seedream 4.0, Nanobanana, Kontext.
Quality up to 4K, flexible aspect ratio options (Square, Portrait, Landscape, Custom).

Responsive & Modern UI:
Light / dark mode, adjustable floating sidebar, central preview, and quick modal for preview & edit.
English and Indonesian Support
Suitable for casual to professional users who need full speed and control.
